Limbajul HTML

HTML (HyperText Markup Language) este un limbaj de marcare pentru a crea pagini web.

Un exemplu de cod html il gasiti in imaginea urmatoare care se afla la adresa https://commons.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.
Imaginea are atributul height de valoare 147.
Tag-urile html cele mai utilizate tabelelor sunt:
table defineste un tabel
tr un rand intr-un tabel
td o celula intr-un tabel
Observatie:Pentru a pune un contur unui tabel se poate( dar nu se recomanda) sa se puna atributul border = "1".
La validare va da eroarea: The border attribute on the table is obsolete.

Formularele se pot crea cu ajutorul tag-ului form care poate contine tag-urile: input, textarea, label, select, option, button.
Pentru mai multe detalii consultati cursul sau tutorialul HTML Forms and Inputs de la adresa https://www.w3schools.com/html/html_forms.asp.

Un exemplu de formular este urmatorul:
Nume:
Prenume:
Parola:
Culoarea preferata:
Descriere:
Sunt de acord cu conditiile

Pentru a crea liste se pot folosi tag-urile:

    i. ul - listă neordonată
    ii. ol - listă ordonată
    iii. li - item din listă
    iv. dl - listă cu descrieri
    v. dt - termen dintr-o listă cu descrieri
    vi. dd - descrierea dintr-o listă cu descrieri

Pentru a insera un document HTML în altul se poate folosi tag-ul <iframe >.
Astfel, exemplul de mai jos inserează conținutul de la adresa https://html.spec.whatwg.org/multipage/.
